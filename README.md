# Detector de Gambiarra

## O que é

O Detector de Gambiarra é uma ferramenta criada para identificar soluções "criativas" que desafiam as boas práticas da programação. Seu objetivo é analisar o código e emitir um alerta sempre que encontrar uma gambiarra em potencial.

O sistema utiliza uma inteligência artificial altamente treinada em projetos legados, códigos feitos às 3 da manhã e commits com a mensagem "agora vai".

## Funcionalidades

- Detecta variáveis com nomes como `teste`, `abc`, `coisa` e `x`.
- Identifica funções com mais de 500 linhas.
- Emite alertas para `if` dentro de `if` dentro de `if`.
- Localiza comentários como `// não mexe aqui`.
- Calcula o Índice Nacional de Gambiarra (ING) do projeto.
- Sugere terapia quando encontra mais de 10 gambiarras consecutivas.

## Como rodar

1. Clone este repositório:
   git clone https://github.com/seu-usuario/detector-de-gambiarra.git

2. Acesse a pasta do projeto:
   cd detector-de-gambiarra

3. Abra o arquivo `index.html` em seu navegador.

4. Aguarde o diagnóstico. Caso o sistema exiba **"Nível de Gambiarra: Crítico"**, recomenda-se fazer um backup antes de continuar.
